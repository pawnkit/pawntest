# Security policy

Report vulnerabilities through GitHub's private
[security advisory form](https://github.com/pawnkit/pawntest/security/advisories/new).
Please do not open a public issue before a fix or mitigation is available.

Test source, AMX files, compiler output, and native providers are untrusted.
Report path escapes, unsafe process execution, unbounded resource use, or cases
where native code runs without explicit opt-in.
